复习到蛋白质了，那么我们首先来聊聊氨基酸吧

# 氨基酸！

# 最基础的两个氨基酸

# 甘氨酸（glycine，Gly）

甘氨酸（Glycine）有特殊甜味，其甜度是蔗糖的80%。所以它的英文名称来自希腊语glykys（甘甜的味道），中文译名既蕴含甘甜之意，又有相似的发音，堪称“信、达、雅”的典范。

因为有甜味，所以甘氨酸在食品工业中经常作为调味剂，起到去苦增甜的作用。所以，有同学说他用“苦尽甘来”一词记忆甘氨酸，倒是很恰当的。

甘氨酸的侧链最小，只有一个氢原子。这使他与众不同，是唯一没有手性的基本氨基酸。小巧灵活是甘氨酸在蛋白质中的特点，比如胶原蛋白的三股螺旋构象是非常特殊的，必须要每隔两个残基就有一个甘氨酸，否则就会导致空间位阻过大。

同理，蛋白质两个结构域之间的连接也经常需要甘氨酸提供构象的灵活性。

不过，甘氨酸灵活有余，稳定就难免不足。在形成α-螺旋的时候，甘氨酸是最强的破坏者之一，原因就是侧链太小，构象根本稳定不下来。另外，甘氨酸常用来配制缓冲液，它作为缓冲剂，在电泳过程中保持 pH 值并防止样品损坏。甘氨酸还用于去除蛋白质印迹膜上的蛋白质标记抗体，以便能够从 SDS-PAGE 凝胶中探测许多感兴趣的蛋白质。经常做电泳实验的同学一定记得。

Pubchem：https://pubchem.ncbi.nlm.nih.gov/compound/Glycine

维基百科：谷氨酸

# 丙氨酸（alanine，Ala）

丙氨酸（alanine，Ala）的英文名字来源于德语乙醛，而中文名字就很好理解，因为丙氨酸含有三个碳，化学名就是氨基丙酸。

这是一个朴实无华的名字，就像氨基酸的特点一样。

丙氨酸的侧链只有一个甲基，比甘氨酸稍大一点。画其它18种氨基酸结构式的时候都是在丙氨酸上添加基团来画的。

在蛋白质中，丙氨酸的作用就像一块砖，平平无奇，最基础的建筑材料，跟谁也不冲突。

丙氨酸的侧链位阻小，放在α-螺旋里，它是最强的形成者之一；放到β-折叠里，也非常稳定。

在蛋白质工程里，如果要把蛋白质上的某个氨基酸突变掉，又没有特定的目标，一般可以突变成丙氨酸，这样不容易破坏蛋白质的整体构象。大多数经典氨基酸可以被视为丙氨酸的化学衍生物。因此，蛋白质中的大多数经典氨基酸可以通过点突变与 Ala 交换，而二级结构保持完整。Ala 模拟大多数编码氨基酸的二级结构偏好这一事实在丙氨酸扫描诱变中被实际利用。此外，经典的X 射线晶体学通常采用聚丙氨酸骨架模型，通过分子置换（一种基于模型的定相方法来确定蛋白质的三维结构。

维基百科：丙氨酸

# 支链氨基酸

支链氨基酸有三种，分别是缬氨酸（valine，Val）、亮氨酸（leucine，Leu）和异亮氨酸（isoleucine，Ile）。他们的侧链都具有分支结构，所以叫支链氨基酸**（Branched Chain Amino Acid，BCAA）**。

# 缬氨酸（valine，Val）

缬氨酸的英文名字Valine来自于缬草酸（Valinic acid），这种酸是从缬草（Valerine）根部提取的，具有安神，理气，止痛作用。那么他们有什么联系吗？看化学名就知道了。缬草酸的化学名是正戊酸，缬氨酸是氨基异戊酸，的确很相似。这个翻译算是很忠实原义了。

维基百科：缬氨酸

# 亮氨酸（leucine，Leu）

亮氨酸最早是从奶酪中提取出来的，其结晶是白色有光泽的六面体，折光度很高，非常闪亮，而且粉末也是白白亮亮的。所以它的英文名字是“Leucine”，来自希腊语Leukos（白）。Leuk是拉丁语前缀“白色的”，比如Leukemia是白血病，leukocyte是白细胞。亮氨酸中文的别名就是白氨酸。

在代谢上，三种支链氨基酸人体都不能自己合成，只能从食物中摄取，所以称为必需氨基酸。支链氨基酸虽然只有三种，但却占肌肉中氨基酸含量的14-18%，它们对肌肉量的调节有很大作用。支链氨基酸（主要是亮氨酸）可以促进蛋白质合成，所以增加亮氨酸摄入可以帮助增肌。健身运动之后可以适当补充亮氨酸或支链氨基酸含量丰富的乳清蛋白，以促进蛋白质合成。

维基百科：亮氨酸

# 异亮氨酸（isoleucine，Ile）

异亮氨酸是亮氨酸的同分异构体，把亮氨酸侧链的异丁基换成了仲丁基。所以它的英文名字叫做isoleucine，iso就是“异”的词头。中文译名也是直译，便于记忆。异亮氨酸也有个中文别名，就是异白氨酸。

在构成蛋白质的时候，三种支链氨基酸都是疏水性的，一般出现在蛋白质的疏水区中。不过，他们还是有一些区别的。缬氨酸和异亮氨酸的分支都在b-碳上，空间位阻比较大，形成a-螺旋的能力要比亮氨酸弱一些。

维基百科：异亮氨酸

# 侧链带有羟基的氨基酸

# 丝氨酸（serine，Ser）

丝氨酸最初是由 Emil Cramer 于 1865 年从蚕丝中获得的，英文名字来自拉丁语sericum（丝），中文译名也算是音义兼顾了。它也有个别名，叫蚕丝氨基酸。丝氨酸结构比较简单，在丙氨酸的侧链甲基上加一个羟基就可以了，其生物合成也简单，所以是非必需氨基酸。

维基百科：丝氨酸

# 苏氨酸（threonine，Thr）

苏氨酸最早是从纤维蛋白中分离出来的，其结构与threose（苏阿糖）相似，都是四个碳并含有羟基，所以命名为threonine。

苏阿糖和苏氨酸

图片来源：https://chemtymology.co.uk/2019/10/13/threonine-threose-and-erythrose/

苏氨酸的结构要复杂一些，比丝氨酸多了一个甲基。就因为多了一个甲基，它的合成路线就完全不同，人体自身无法合成，所以苏氨酸是必需氨基酸。

维基百科：苏氨酸

# 丝氨酸和苏氨酸羟基的作用

丝氨酸和苏氨酸分子中的羟基，对蛋白质的功能非常重要。比如，它已被证明存在于胰凝乳蛋白酶、胰蛋白酶和许多其他酶的活性位点中。

胰蛋白酶的活性中心含有丝氨酸

有机磷化合物可与酶活性中心丝氨酸上的羟基牢固结合，从而抑制其活性。此类化合物强烈抑制胆碱酯酶，使乙酰胆碱堆积，引起一系列神经中毒症状包括肌肉痉挛、神经麻痹等，称为神经毒剂。二战中使用的DFP，邪教组织奥姆真理教在东京地铁中使用的沙林，都属于此类。

丝氨酸和苏氨酸还在蛋白质的糖基化以及磷酸化过程中起重要作用，研究蛋白质修饰、信号通路的人对它们都很熟悉。

丝氨酸、苏氨酸是在最常见的磷酸化氨基酸之二【剩下一个是酪氨酸（酪氨酸也有一个羟基）】。磷酸化的丝氨酸残基通常被称为磷酸丝氨酸或丝氨酸磷脂。

磷酸化

磷酸化修饰可以发生在多种氨基酸残基上，可分为四大类：1.丝氨酸、苏氨酸、酪氨酸、羟脯氨酸羟基残基上的O-磷酸化；2.组氨酸、赖氨酸残基上的N-磷酸化；3.半胱氨酸残基上的S-磷酸化；4.天冬氨酸、谷氨酸残基上的Acyl-磷酸化；

在真核细胞中，以丝氨酸（S）、苏氨酸（T）和酪氨酸（Y）残基的磷酸化修饰为主，因此蛋白质激酶被分类为丝氨酸/苏氨酸激酶、酪氨酸激酶或双作用激酶。三种氨基酸的磷酸化在同一个细胞中相对丰度比例约为pS:pT:pY= 1800:200:1

同时丝氨酸和苏氨酸的羟基还可以发生O-糖基化。O-糖链与蛋白质的丝氨酸或苏氨酸的自由OH基共价连接。O-糖基化位点没有保守序列,糖链也没有固定的核心结构,组成既可是一个单糖,也可以是巨大的磺酸化多糖,因此与N-糖基化相比,O-糖基化分析会更加复杂。

O-糖基化在高尔基体中进行,通常第一个连接上去的糖单元是N-乙酰半乳糖,连接的部位为Ser、Thr或Hyp的羟基,然后逐次将糖残基转移上去形成寡糖链,糖的供体同样为核苷糖,如UDP-半乳糖。O-糖蛋白主要存在于黏液和免疫球蛋白等。

# 侧链带硫的极性氨基酸

# 甲硫氨酸（methionine，Met）

甲硫氨酸的英文名字methionine来自甲基（methyl）和硫基（thio），这两个词头都是来源于希腊语。中文译名很直接，也有利于记忆缩写符号：Me代表甲基，t代表硫。

甲硫氨酸的别名叫蛋氨酸，是因为它在卵白蛋白中含量很高。另外，甲硫氨酸分解的时候会产生硫化氢，所以有臭鸡蛋味。

甲硫氨酸在体内可以作为通用甲基供体（通过形成S-腺苷甲硫氨酸），参与多种分子的甲基化反应。

各种慢性肝损伤都会导致肝脏S-腺苷甲硫氨酸合成减少，所以医生经常建议补充甲硫氨酸保护肝脏。甲硫氨酸的流失与头发的老年性变白有关。它的缺乏导致过氧化氢在毛囊中积聚，酪氨酸酶有效性降低，头发颜色逐渐消失。

提供甲基也是甲硫氨酸降解的开始，下一步就是把剩下的硫拿来合成半胱氨酸。所以，半胱氨酸是非必需氨基酸。其实，这个非必需很有些投机取巧的意思，因为要消耗一个必需氨基酸才行。

INFO

类似的还有酪氨酸，合成它要消耗一个必需的苯丙氨酸。

所以，有人把它们叫做半必需氨基酸或条件必需氨基酸。因为可以转化为半胱氨酸，所以甲硫氨酸也具有抗氧化和解毒的功能。甲硫氨酸剩下的部分最后变成琥珀酰辅酶A，可以去进行糖的异生。所以它是生糖氨基酸。人体不能合成甲硫氨酸，所以它是必需氨基酸。

一般来说，蛋白质的合成都离不开甲硫氨酸，因为它的密码子（ATG）是起始密码子。甲硫氨酸的转运RNA也与众不同：原核生物有两种对应的tRNA，真核生物虽然只有一种，但却没有TΨC序列。

此外由于侧链硫原子和芳香族氨基酸之间的S/π 相互作用的稳定性影响，硫醚确实具有较小的结构作用。甲硫氨酸残基不具有催化作用。这与半胱氨酸残基形成对比，其中半胱氨酸的硫醇基团在许多蛋白质中具有催化作用。

维基百科：甲硫氨酸

# 半胱氨酸（cysteine，Cys）

半胱氨酸顾名思义，就是胱氨酸的一半。

胱氨酸（Cystine）是在从**膀胱结石（cystolith）**中发现的，所以命名为胱氨酸。两个半胱氨酸被氧化，通过二硫键连接到一起，就形成了胱氨酸。所以半胱氨酸是还原剂，在蛋白质研究中可以用来打开二硫键，或保持某些蛋白活性中心的巯基处于还原状态。半胱氨酸在食品上可以作为抗氧化剂，添加在果汁中，可以保护维生素C，还可以防止果汁变成褐色。半胱氨酸在生物体中的还原作用主要是由于它可以合成谷胱甘肽。

在蛋白质结构中，半胱氨酸可以通过在分子间形成形成二硫键来维持蛋白质高级结构。一般维持蛋白质构象的都是次级键，所以二硫键是很特殊的。所以二硫键改变引起的失活一般也看作变性。

对蛋白质的功能来说，半胱氨酸的贡献主要就是活泼的巯基(-SH)了。很多酶的活性中心都有巯基，可以起到亲核催化和一般酸碱催化的作用。

半胱氨酸的巯基可以和多种金属离子结合，重金属中毒的原因之一就是使巯基酶失活。半胱氨酸能置换出酶中的巯基，故可作为重金属中毒的解毒剂。烷化剂（如碘乙酸等），可以烷化巯基，使酶失活。有机砷、汞化合物也是通过与巯基作用，来抑制含巯基的酶。

维基百科：半胱氨酸，其中Roles in protein structure讨论了Cys在蛋白质结构中的作用。

# 侧链带羧基的氨基酸

# 天冬氨酸（asparatic acid，Asp）

天冬氨酸是从百合科天门冬属植物（asparagus）中发现的，所以也称为天门冬氨酸。

天冬氨酸参与尿素循环，可以降低血液中氨和二氧化碳的量，所以经常作为氨解毒剂和疲劳恢复剂。天冬氨酸在食品工业上用于合成甜味剂，因为它是阿斯巴甜（天冬氨酰苯丙氨酸甲酯）的主要原料。也可作为饮料的添加剂，用于补充电解质。

维基百科：天冬氨酸

# 谷氨酸（glutamic acid，Glu）

谷氨酸是从海带中发现的，但是谷氨酰胺却是四十多年前从小麦面筋（gluten）中提取出来的，因为主要来源于谷蛋白（glutelin）而得名。谷氨酸具有鲜味，谷氨酸钠是味精的主要成分。其实，天冬氨酸也有鲜味，但比谷氨酸弱一些。

谷氨酸的侧链比天冬氨酸多出一个亚甲基，这导致它可以与自身的氨基反应，形成焦谷氨酸。在生活中，这是味精不能在高温下使用的原因。

在蛋白质中，这个反应可以封闭肽链的末端氨基。如果一条肽链只有羧基末端，却测不出氨基端是什么，那么很可能是生成了焦谷氨酸。

在蛋白质中，它们都可以为蛋白质表面提供负电荷。另外，天冬氨酸经常出现在酶的活性中心，可以参与构成质子传递通路（如胰蛋白酶），也可以直接起催化作用（如HIV蛋白酶）。

维基百科：谷氨酸

# 天冬酰胺（asparagine，Asn）

天冬酰胺（asparagine，Asn）和谷氨酰胺（glutamine，Gln）可以看作上面两种酸性氨基酸的酰胺衍生物，在体内很容易与对应的氨基酸互相转换。

在蛋白质酸水解时，这两种氨基酸都会被水解成对应的酸。

在糖蛋白中，天冬酰胺可用于连接糖链，因为天冬酰胺侧链上的酰胺基团可以和糖链上的半缩醛羟基反应，形成N-糖苷键。

当加热到足够的温度时，天冬酰胺与还原糖或其他羰基化合物之间的反应会在食物中产生丙烯酰胺（Acrylamide）。这些产品出现在炸薯条、薯片和烤面包等烘焙食品中。国际癌症研究机构（IARC）将丙烯酰胺分类为可能的人类致癌物。

由于天冬酰胺侧链可以与肽主链形成氢键相互作用，因此天冬酰胺残基通常作为asx转角和asx 基序在 α 螺旋的开头附近，并且在类似的转角基序中，或作为酰胺环，在 β 折叠中。它的作用可以被认为是“加盖”氢键相互作用，否则多肽主链会满足这些相互作用。

天冬酰胺还提供了N-连接糖基化关键位点，通过添加碳水化合物链来修饰蛋白质链。通常，如果天冬酰胺残基的 C 侧侧翼为 X-丝氨酸或 X-苏氨酸，其中 X 是除脯氨酸之外的任何氨基酸，则碳水化合物树可以单独添加到天冬酰胺残基上。

维基百科：天冬酰胺

# 谷氨酰胺（glutamine，Gln）

谷氨酰胺是生物体最重要的氨基供体，多种氨基酸和碱基的生物合成都需要谷氨酰胺。所以，谷氨酰胺具有促进蛋白质与核酸合成的作用，对于快速生长和增殖的组织和细胞，如粘膜细胞、淋巴细胞、内皮细胞和肿瘤细胞等十分重要。

谷氨酰胺是哺乳动物体内含量最高的氨基酸，占人体全部游离氨基酸的一半以上。这与它的氨转运功能有一定关系。因为氨具有神经毒性，所以不能以游离状态在血液中运输，所以生成无害的谷氨酰胺作为运输形式。

对于大运动量的人，尤其是耐力运动者，额外补充谷氨酰胺有助于训练后的恢复、提高免疫力。健身者一般都会补充谷氨酰胺以促进蛋白质合成。在肿瘤治疗方面，谷氨酰胺代谢已经成为一个重要的抗癌靶点。许多针对谷氨酰胺代谢的化合物成为了抗癌药物研究的热点。

在疾病或高强度运动等应激状态下，机体对谷氨酰胺的需求量增加，自身合成不能满足需要，就需要从外界摄取。所以，谷氨酰胺是一种条件必需氨基酸。

维基百科：谷氨酰胺

# 侧链其他集团的极性氨基酸

在基本氨基酸中，碱性氨基酸有三种，这里介绍属于脂肪族的两种：精氨酸（arginine ，Arg）和赖氨酸（lysine，Lys），组氨酸因为含有杂环，所以放在最后讲。

# 精氨酸（arginine ，Arg）

精氨酸最初是从羽扇豆苗中分离提取的。精氨酸的硝酸盐是银白色的，所以被用希腊语argiros（银白色的金属）命名。中文译为精氨酸是因为它大量存在于鱼精蛋白(protamine)中。可能译者觉得把它译为“银白氨基酸”实在不妥，就干脆另起炉灶了。

精氨酸侧链上的胍基碱性较强，所以精氨酸是碱性最强的氨基酸，等电点是10.76。精氨酸通常存在于蛋白质的外部，其中亲水性头基可以与极性环境相互作用，例如参与氢键和盐桥。

精氨酸参与尿素循环，可促进尿素的生成，所以可以用来治疗氨中毒造成的肝昏迷。精氨酸也是精子蛋白的主要成分，有促进精子生成，提高精子运动能量的作用。精氨酸还是一氧化氮合酶（NOS）的底物。一次服用精氨酸不能超过5g，过量摄取会导致急性胰腺炎。

精氨酸参与多胺合成。多胺带有多个正电荷，可与DNA结合，促进细胞生长，现在已经成为设计抗癌新药的重要靶标。精氨酸可通过尿素循环合成，所以它不是必需氨基酸。但它对婴儿和某些病人是必需的，所以称为半必需或条件必需氨基酸。

维基百科：精氨酸

# 赖氨酸（lysine，Lys）

赖氨酸最初是从干酪素水解物分离出来的，被命名为lysin，有裂解物的意思。赖氨酸是单纯的音译。赖氨酸是必需氨基酸，而且赖氨酸在谷类蛋白质中含量少,是谷类蛋白质的第一限制氨基酸。

限制氨基酸

指食物蛋白质中缺乏最严重的必需氨基酸，会限制此种蛋白质的营养价值。

在吸收消化利用过程中，氨基酸需要一定的比例才能被充分的吸收利用。当某食物中某氨基酸远远不能达到这个比例时，即使蛋白质含量再高，也发挥不出他的优势，这个氨基酸就是限制性氨基酸。比如豆类中的甲硫氨酸（蛋氨酸）和谷类中的赖氨酸。

赖氨酸对于蛋白的功能非常重要，除了提供正电荷之外，它独特的结构也很有用。赖氨酸侧链末端的氨基是亲水的，但烃链部分又是疏水的。所以它可以出现在蛋白质的一些两亲结构中。

赖氨酸有一条长长的侧链，可以从蛋白质中伸出。生物素、硫辛酸等带有羧基的小分子，都可以通过与赖氨酸的侧链氨基形成酰胺键来与蛋白质形成牢固的共价连接。在丙酮酸脱氢酶复合体中，赖氨酸长长的侧链和硫辛酰胺一起构成一条“手臂”，在电荷的推动下携带中间产物移动。

组蛋白中的赖氨酸经常被各种修饰，比如甲基化、乙酰化、泛素化、苏木化等等，这些修饰与基因表达调控、染色质高级结构变化等生物过程密切相关，做表观遗传的同学会很熟悉的。此外，胶原蛋白的交联也需要赖氨酸参与。赖氨酸还被用来合成肉碱，而肉碱是脂肪酸进入线粒体氧化分解所必需的。

赖氨酸的分解途径比较独特，它不参与转氨反应。赖氨酸一般是先脱去侧链氨基，生成氨基己二酸，然后再转氨。

维基百科：赖氨酸

# 芳香族氨基酸

芳香族氨基酸有三种：苯丙氨酸（phenylalanine，Phe）、酪氨酸(tyrosine，Tyr) 和色氨酸(tryptophan，Trp)。其实，如果按照化学上的定义，组氨酸的侧链咪唑基属于芳杂环。所以组氨酸应该是既属于芳香族氨基酸，也属于杂环氨基酸。就是说，芳香族氨基酸一共有四种，其中三种含苯环，一种不含苯环。

维基百科上就把组氨酸算作芳香族氨基酸。不过生化书一般还是习惯于把组氨酸算作杂环氨基酸，只把三个带有苯环的氨基酸算作芳香族氨基酸。【《生物化学，第三版》王境岩，杨荣武《生物化学原理》，《Lehninger Principles of Biochemistry 7th》中芳香族氨基酸都不包括组氨酸】

# 苯丙氨酸（phenylalanine，Phe）

苯丙氨酸的英文名中“phenyl”就是苯基，alanine就是丙氨酸，跟中文一样好记。它的结构式也是丙氨酸加上苯基，这就是用化学结构命名的好处。

苯是疏水的，所以苯丙氨酸疏水性很强，一般出现在蛋白质的疏水区中。苯丙氨酸有紫外吸收，吸收峰在257纳米，摩尔消光系数仅为200，又经常埋藏在蛋白质内部，所以对蛋白质的紫外吸收贡献不大。苯丙氨酸人体不能合成，所以是必需氨基酸。

维基百科：苯丙氨酸

# 酪氨酸(tyrosine，Tyr)

酪氨酸是首先在乳酪中发现的，**“Tyro”**在希腊文中的意思就是乳酪。

酪氨酸侧链的酚基是一个可解离基团，但解离常数较高（pKa=10.1），所以在中性条件下基本不带电荷。酪氨酸的紫外吸收较强，吸收峰在275纳米，摩尔消光系数1400。

酪氨酸在人体内可以合成多种生物活性物质，如黑色素、肾上腺素、甲状腺素等。切开的苹果、土豆变色也是由于酪氨酸被氧化生成有色物质。

皮肤变黑

黑素在黑素细胞内合成，经过树突状黑素细胞转移到临近的角质形成细胞，后随着角质形成的成熟而弥散到表皮，并随着角质形成细胞的脱落而与表皮分离。

皮肤当中含有较多的**“黑素细胞”，“黑素细胞”将酪氨酸当做原料，再经过酪氨酸酶的催化，经过一系列反应最终产生“黑素”**。“黑素”是我们皮肤颜色的最终决定因素，而黑素体的生成及黑素化过程中，酪氨酸酶的活性起着关键的作用。

人体表皮内抑制酪氨酸酶活性的有机化合物-巯基，以谷胱甘肽为代表，正常情况下巯基与酪氨酸酶互相制约，皮肤肤色正常。巯基因氧化而减少时则黑色素生成增多。

酪氨酸是非必需氨基酸，因为人体可以用苯丙氨酸合成酪氨酸。这种合成需要消耗一个必需氨基酸，所以也像半胱氨酸那样被称为半必需氨基酸。

维基百科：酪氨酸

# 色氨酸(tryptophan，Trp)

色氨酸是用胰蛋白酶消化酪蛋白时分离得到的，所以它的名称来源于trypsin（胰蛋白酶），和phane（希腊语“出现”）。色氨酸侧链的吲哚基是很强的发色团，摩尔消光系数高达5600，可能是中文命名的来源。蛋白质的紫外吸收主要就是由色氨酸和酪氨酸决定的。吲哚基也属于芳杂环，所以色氨酸也属于杂环氨基酸。色氨酸人体不能合成，所以是必需氨基酸。

色氨酸除了用于合成蛋白质之外，也用于合成一些生物活性物质。

  • 在植物中，色氨酸是生长素吲哚乙酸的前体。
  • 在动物体内，色氨酸是神经递质5-羟色胺(5-HT)的前体，五羟色胺具有解除疲劳、促进睡眠的作用，缺乏5-羟色胺容易抑郁
  • 烟酸，也称为维生素 B 3，是由色氨酸通过犬尿氨酸和喹啉酸合成的。
  • 色氨酸分解的中间物可转变为尼克酸，但合成率很低，所以仍然需要从食物中摄取尼克酸。

维基百科：色氨酸

# 杂化氨基酸

20个基本氨基酸中，杂环氨基酸有3种：色氨酸、组氨酸和脯氨酸。色氨酸和组氨酸都含有芳杂环，但色氨酸含有苯环，有很强的紫外吸收，所以放在芳香族氨基酸中讲过了。所以今天介绍剩下的两个杂环氨基酸：组氨酸(histidine，His)和脯氨酸(proline，Pro)。

# 组氨酸(histidine，His)

组氨酸的英文名histidine来自于德语histidin。histo在希腊语中是组织的意思。histo也是拉丁语前缀，组织或网的意思。很多histo开头的医学生物单词，都翻译成“组”，例如组胺histamine，组蛋白histone，免疫组化immunohistochemistry。所以中文译为组氨酸是顺理成章的。

组氨酸可以脱羧形成组胺，组胺具有促进平滑肌舒张、微血管扩张、胃酸分泌等作用。所以组胺酸可以降低血压，临床上用于心绞痛、心功能不全等疾病的治疗；也可以引起支气管哮喘、丘疹等过敏反应，临床用抗组胺药物治疗过敏。组胺酸虽然也属于芳香族氨基酸，但它对蛋白质在280纳米的紫外吸收没有明显贡献。因为咪唑的吸收峰在209.1nm，在280nm有吸收，但强度较弱。不过，如果是高浓度的组胺酸或咪唑，对紫外吸收还是有影响的。在用镍柱纯化融合蛋白的时候可以注意一下。

组氨酸的侧链含有咪唑基，在中性条件下带正电荷，所以是碱性氨基酸。不过它的碱性较弱，等电点是7.59。这是一个非常有用的特性。比如说，这使组氨酸在血液中具有较强的缓冲能力。组氨酸在生理条件下是否带电，与周围内环境有很大关系。

所以蛋白质中的His残基可以扮演酸碱两性基团，因此常是酶中发生酸碱催化机理的部分经常出现在酶的活性中心，起一般酸碱催化作用。咪唑基提供质子和接受质子的速度都很快，所以也经常用来构成质子传递链。

除了参与催化，组氨酸还能与铁等金属离子配位。比如血红素与肌红蛋白或血红蛋白的连接，就是通过组氨酸与铁形成配位键。组氨酸能与铁或其他金属离子形成配位化合物，还可以促进铁的吸收，因而可用于防治贫血。

组氨酸可发生脱羧生成组胺，这也是一种常见的生物分子，具有降低血压，收缩子宫等功能。细胞释放的组胺常是导致荨麻疹的原因之一。

人类可以合成组氨酸，所以它不是必需氨基酸。但是，幼龄动物和婴儿体内的组氨酸合成量不能满足机体生长需要，所以它是半必需氨基酸。

维基百科：组氨酸

# 脯氨酸(proline，Pro)

脯氨酸是在白明胶中首先发现的，学名为Pyrrolidone Carboxylic Acid（吡咯烷酮羧酸），简称为“Proline”。中文译为“脯氨酸”是因为白明胶是用动物皮肤、肌腱等结缔组织中的胶原蛋白熬制得到的部分水解产物，“脯”就是肉干的意思。

脯氨酸的结构也是很特别的。一方面，它不像色氨酸和组胺酸那样属于芳杂环，它属于脂肪族，所以是脂杂环。另一方面，它是基本氨基酸中唯一的亚氨基酸。换句话说，其它氨基酸都是伯胺，只有它是仲胺（氮原子与两个碳原子相连）。脯氨酸侧链独特的环状结构使脯氨酸与其他氨基酸相比具有特殊的构象刚性。这种不同使它具有一些特性，比如不参加转氨，限制肽链构象等。

脯氨酸的特殊构象对于胶原蛋白的三股螺旋是必需的，但对于a-螺旋却是破坏者。因为脯氨酸的氮原子与侧链形成了环，所以Cα－N键不能自由旋转，无法达到a-螺旋构象要求的二面角。因此，a-螺旋延伸到脯氨酸就会终止。这与甘氨酸的过度自由旋转恰好是两个极端。另外，脯氨酸在合成蛋白质的时候，有时还会需要一个肽基脯氨酰顺反异构酶（PPI）的辅助。这个酶表达量非常恒定，是一个检测基因表达量的良好内参。

维基百科：脯氨酸
